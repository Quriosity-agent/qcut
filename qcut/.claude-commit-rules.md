# Commit Guidelines for QCut Project

## Strict Rules
1. **Single-line commits only** - Keep commit messages concise and to the point
2. **NO attribution to Claude or any AI tool** - Never include 'Generated by Claude' or any co-author information
3. No multi-line commit messages
4. No emoji in commit messages unless specifically requested

## Example Good Commits
- Fix CSP blob URL violations for WaveSurfer audio
- Add console error fix documentation
- Optimize AudioWaveform component performance
- Fix maximum update depth warning in timeline

## Example Bad Commits (DO NOT USE)
- Any message with 'Generated by Claude'
- Any message with 'Co-Authored-By'
- Multi-line messages with descriptions
- Messages with AI tool attribution
